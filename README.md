# Şef Ekipman Yükseltme Hesaplayıcısı

## Özellikler
- Şef Donanım Yükseltme Hesaplayıcısı
- Şef Tılsım Yükseltme Hesaplayıcısı  
- Çift dil desteği (Türkçe/İngilizce)
- Responsive tasarım

## Kullanım
1. `index.html` dosyasını indirin
2. Herhangi bir tarayıcıda açın
3. Sol menüden hesaplayıcı seçin
4. Seviyeleri ayarlayıp HESAPLA butonuna tıklayın
